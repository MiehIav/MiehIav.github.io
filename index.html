 Usage notes & quick examples

This dashboard is a curated link hub. Below are example fetch snippets (replace YOUR_KEY and coordinates as needed). For no-key options, consider open-meteo.com or MET Norway (see above).

// Example: fetch Open-Meteo (no API key)
fetch('https://api.open-meteo.com/v1/forecast?latitude=49.8951&longitude=-97.1384&hourly=temperature_2m,relativehumidity_2m')
  .then(r => r.json())
  .then(data => console.log(data));

// Example: fetch OpenWeatherMap (requires API key)
fetch('https://api.openweathermap.org/data/2.5/onecall?lat=49.8951&lon=-97.1384&units=metric&appid=YOUR_KEY')
  .then(r => r.json()).then(d => console.log(d));

Tip: When using MET Norway (api.met.no), set a descriptive User-Agent header to avoid 403 errors (their docs require it).
